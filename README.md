🏠 Real-State-Analysis
A complete analysis of real estate data from Gurgaon using Python libraries such as Pandas, NumPy, Seaborn, and Matplotlib. This project demonstrates data cleaning, preprocessing, and visualization to uncover insights from raw CSV files.

📌 Project Overview
This repository contains:

Raw dataset: Gurgown_Real_Estate.csv

Cleaned dataset: Gurgown_Real_Estate_Cleaned.csv

Analysis script: RealEstate.py

Visualizations: Graphs and plots generated using Seaborn & Matplotlib

The goal is to transform messy real estate data into meaningful insights by applying data cleaning techniques and exploratory data analysis (EDA).

⚙️ Features
Data cleaning (handling missing values, duplicates, formatting issues)

Exploratory Data Analysis (EDA) with Pandas & NumPy

Visualizations (price trends, distribution plots, correlations)

Export of cleaned dataset for further use

🚀 Getting Started
Prerequisites
Ensure you have Python 3.8+ installed. Install required libraries:

bash
pip install pandas numpy seaborn matplotlib
Running the Analysis
Clone the repository:

bash
git clone https://github.com/ShafAnsh/Real-State-Analysis.git
Navigate to the project folder:

bash
cd Real-State-Analysis
Run the script:

bash
python RealEstate.py
